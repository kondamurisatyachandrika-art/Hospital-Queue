# Hospital Management Queue System in C++

## Project Overview
This is a simple Hospital Queue Management System developed using C++ and Priority Queue Data Structure.

The system manages patients based on priority levels:
- Emergency Patients
- Serious Patients
- Normal Patients

Patients with higher priority are served first.

---

# Features

- Add Patient
- Serve Patient
- Display All Patients
- Priority-Based Queue Management
- Menu Driven Console Application
